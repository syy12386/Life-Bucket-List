# 人生已完成清单 (Life Bucket List)

一个基于 Vue 3 开发的交互式人生清单应用，帮助用户记录和追踪人生中的各种体验和成就。

## 功能特点

- ✅ **标签管理**：添加、删除、编辑标签，标记已完成的人生体验
- 🎨 **主题切换**：支持浅色和深色主题
- 📊 **进度可视化**：实时显示完成进度和统计数据
- 🏆 **成就系统**：根据完成的体验解锁不同成就
- 🎯 **随机挑战**：随机推荐未完成的体验
- 📷 **分享功能**：生成清单截图分享给朋友
- 🔊 **背景音乐**：内置舒缓的背景音乐，可控制开关
- 💾 **数据持久化**：使用 localStorage 保存数据，刷新页面后数据不丢失
- 🎭 **开场动画**：精美的开场文案和动画效果
- ✨ **平滑过渡**：标签状态切换时的平滑动画效果

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **样式**：原生 CSS
- **第三方库**：html2canvas（用于生成分享截图）
- **数据存储**：localStorage

## 使用指南

### 标记体验

- **点击标签**：切换标签的完成状态
- **双击标签**：编辑标签文本
- **点击删除按钮**：删除标签（标签 hover 时显示）

### 添加新体验

在页面底部的输入框中输入新的体验，然后点击「添加」按钮或按回车键。

### 过滤体验

使用页面底部的过滤按钮，可以按「全部」、「已完成」或「未完成」筛选标签。

### 重置清单

点击「重选」按钮，可以重置所有标签的完成状态和成就。

### 随机挑战

点击「随机挑战」按钮，系统会随机推荐一个未完成的体验。

### 查看成就

点击页面顶部的「成就」按钮，可以查看已解锁和未解锁的成就。

### 分享清单

点击「生成截图」按钮，系统会生成当前清单的截图，你可以保存并分享给朋友。

### 控制音乐

点击页面左上角的音乐图标，可以控制背景音乐的播放和暂停。
